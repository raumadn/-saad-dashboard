# 5.5 Inch Phone Version

Dashboard ini sudah disesuaikan untuk layar handphone sekitar 5.5 inch.

Perubahan:
- Sidebar disembunyikan total.
- Header dibuat lebih pendek.
- KPI card diperkecil.
- Globe Plotly diperkecil.
- Tinggi chart diperkecil.
- Tab dibuat horizontal-scroll.
- Padding dashboard dibuat lebih rapat.
- Tombol download dibuat full-width.

Setelah upload ke GitHub:
1. Commit changes.
2. Streamlit Cloud → Manage app → Clear cache.
3. Reboot app.
